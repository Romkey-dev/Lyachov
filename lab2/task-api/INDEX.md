### Исходный код

**src/server.js** (25 строк)
- Точка входа приложения
- Загрузка переменных окружения
- Запуск сервера на PORT
- Обработка ошибок процесса
- Логирование запуска

**src/app.js** (65 строк)
- Создание Express приложения
- Подключение middleware:
  - helmet (безопасность)
  - cors (cross-origin)
  - rate-limit (ограничение)
  - express.json (парсинг)
- Подключение роутов
- Обработка 404 и ошибок

**src/routes/tasks.js** (280 строк) ⭐ ГЛАВНЫЙ ФАЙЛ
1. GET / - получить все (с фильтрацией, сортировкой, пагинацией)
2. GET /:id - получить одну задачу
3. POST / - создать задачу (с валидацией)
4. PUT /:id - обновить задачу (с валидацией)
5. PATCH /:id/complete - отметить выполненной
6. DELETE /:id - удалить задачу
7. GET /stats/summary - статистика
8. GET /search/text - поиск (полнотекстовый)

**src/middleware/validation.js** (120 строк)
- createTaskSchema (Joi)
  - title: 3-100 символов
  - description: < 500 символов
  - category: enum (work, personal, shopping, health)
  - priority: 1-5
  - dueDate: дата в будущем
- updateTaskSchema (все поля опциональны)
- validateCreateTask middleware
- validateUpdateTask middleware
- validateId middleware

**src/middleware/errorHandler.js** (35 строк)
- notFoundHandler - для 404 ошибок
- errorHandler - централизованная обработка
- Логирование ошибок
- Структурированные JSON ответы

**src/utils/fileOperations.js** (50 строк)
- initializeDataFile() - создать задачи.json
- readData() - читать из файла
- writeData() - писать в файл
- getNextId() - инкрементировать ID
- Обработка ошибок файловой системы

## Быстрая навигация по коду

```
Создание задачи:
└─ src/routes/tasks.js → router.post('/')
   └─ src/middleware/validation.js → validateCreateTask
      └─ src/utils/fileOperations.js → getNextId() + writeData()

Получение задач:
└─ src/routes/tasks.js → router.get('/')
   └─ Фильтрация → tasks.filter()
   └─ Сортировка → tasks.sort()
   └─ Пагинация → tasks.slice()

Поиск задач:
└─ src/routes/tasks.js → router.get('/search/text')
   └─ tasks.filter() → title + description

Статистика:
└─ src/routes/tasks.js → router.get('/stats/summary')
   └─ Подсчет метрик → JSON ответ

Обработка ошибок:
└─ src/middleware/errorHandler.js
   └─ errorHandler middleware
      └─ res.status(err.status).json(error)
```
